# OS Development

Attempt at creating an Operating System from scratch.

## How to run

1. Install `qemu`, `nasm`
2. Build a [C Cross Compiler](https://wiki.osdev.org/GCC_Cross-Compiler)
3. Update **CC* and **GDB** in the `Makefile`
4. Run with `make run`
